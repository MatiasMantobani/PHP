----------------- Crear nuevo Proyecto ------------------------------------------------------------------------------------
XAMPP Control Panel -> Run as adminsitrator
Start Apache

En htdocs creo carpeta proyectos y PHP: proyectos\PHP\
Abro carpeta con VS code -> New File... -> "index.php"
